# r-cat
a rust-ver netcat compative tool. Read and write data across networks - arbitrary TCP and UDP connections and listens.

Syntax
```
nc [-46AcDCdhklnrtUuvz] [-b boundif] [-i interval] [-p source_port]
    [-s source_ip_address] [-w timeout] [-X proxy_protocol]
        [-x proxy_address[:port]] [hostname] [port[s]]
```
